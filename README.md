# MAPID Laravel Geospatial Web App

A responsive geospatial web application built with Laravel 13, Tailwind CSS, and MapLibre GL. This project was developed as part of the MAPID Internship Study Case.

## 🚀 Overview

This application consists of two main pages:

- Portfolio Landing Page
- Interactive Geospatial Map

The landing page showcases personal information, technical skills, and selected projects, while the map page demonstrates geospatial visualization using MapLibre GL and OpenStreetMap.

---

## ✨ Features

### Portfolio Landing Page

- Responsive design using Tailwind CSS
- SEO-friendly structure
- Modern and clean user interface
- Project showcase section
- Reusable Blade Components

### Interactive Geospatial Map

- Interactive map powered by MapLibre GL
- OpenStreetMap basemap
- Point visualization using markers
- LineString visualization
- Polygon visualization
- Navigation controls (zoom and pan)

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/rifqyfakhryzain/mapid-geospatial-app.git
```

Navigate to project directory:

```bash
cd mapid-laravel-geospatial-app
```

Install PHP dependencies:

```bash
composer install
```

Install Node.js dependencies:

```bash
npm install
```

Copy environment file:

```bash
cp .env.example .env
```

Generate application key:

```bash
php artisan key:generate
```

Run development server:

```bash
php artisan serve
```

Start Vite:

```bash
npm run dev
```
